# Units

Units are phases of the course, following the **Tools → Skills → Artifacts** progression.

---

## Course Progression

```
┌─────────────────────┐    ┌─────────────────────┐    ┌─────────────────────┐
│   Unit 1: Tools     │───▶│  Unit 2: Skills     │───▶│ Unit 3: Artifacts   │
│                     │    │                     │    │                     │
│ Environment Setup   │    │ Research Skills     │    │ Contribution        │
│ (Classes 01-02)     │    │ (Classes 03-04)     │    │ (Classes 05+)       │
└─────────────────────┘    └─────────────────────┘    └─────────────────────┘
```
