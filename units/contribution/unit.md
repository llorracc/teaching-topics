# Unit: Contribution

## Overview

This unit focuses on creating artifacts and contributing to open-source repositories: adding yourself as coauthor, improving notebook content, and creating pull requests.

By the end of this unit, you should have:
- Added yourself as coauthor to your ballpark notebook
- Created a pull request to the econ-ark/ballpark repository
- Improved your notebook content using AI assistance
- Received and responded to PR review feedback
