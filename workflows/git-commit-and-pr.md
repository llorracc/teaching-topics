# Workflow: Git add, commit, and PR

**Purpose:** Stage your changes, commit them, and create a pull request.

---

## Steps

### A. Stage and commit

1. Ask Cursor:
   > "Git add and commit my changes."
2. Review and confirm the commit.

### B. Create a pull request

1. Ask Cursor:
   > "Create a pull request for my modifications to the ballpark repo."
2. Follow the instructions to open a PR (this may involve pushing to your fork and opening a PR on GitHub).
