# Workflow: Add LitMap image and subsequent-literature bib

**Purpose:** Add visualization and bibliography for papers that cite your paper.

---

## Steps

1. **Download the LitMap image** you created (showing papers that cite your paper).
2. **Add the image** to the MyST version of the document.
3. **Download the BibTeX file** for the subsequent literature (papers citing your paper) from LitMaps.
4. Place that bib file in your ballpark directory and update MyST config if needed.
