## Learning objectives

- **SolvingMicroDSOPs (SMD):** Orient to the public lecture-note repo and how it defines dynamic stochastic optimization problems in modular form.
- **Sections 12–13 (preview):** See multi-control problems and the **stage / perch** vocabulary (arrival, decision, continuation) that the next assignment will use in depth.
- **Matsya:** Watch the instructor use **Matsya** to navigate or explain structure in SMD-linked materials.
- **Worked examples:** Connect SMD concepts to the **Imai–Keane**-style decomposition notebooks used as templates for the Bellman assignment.
- **Environment:** Everyone can **pull latest SMD**, reinstall if needed, and open notebooks locally.

## In-class workflow

### 1. Sync SolvingMicroDSOPs-Public (~10–15 min)

- Students **pull** (or clone) the latest **SolvingMicroDSOPs-Public** / **SolvingMicroDSOPs** repo.
- Reinstall **environment / dependencies** if the update requires it.
- Confirm a common **commit or tag** if the instructor pins a version for class.

### 2. Q&A — SMD content (~20–30 min)

- Open floor for questions on notation, stages, shocks, and how the lecture notebook is organized.
- Clarify what students **must** read for the next assignment vs optional deep dives.

### 3. SMD lecture notes — finish core discussion (~25–35 min)

- Complete discussion of the planned segments of the **SMD lecture notes** (including the path toward **multiple controls** and **modular stages** as setup for Class 07).
- Tie examples to **economic meaning** (states, controls, timing) rather than only code cells.

### 4. Demo — Matsya on SMD materials (~15–20 min)

- Live demo: **Matsya** driving exploration or explanation using the **SMD** lecture notes (navigation, structure, or instructor workflow—per that day’s build).

### 5. Demo — Matsya on Imai–Keane (~15–20 min)

- Second demo: **Matsya** on **Imai–Keane** decomposition notebooks (one-stage vs two-stage), emphasizing how **perches** and **stage composition** mirror the assignment template.

### 6. Wrap-up (~5–10 min)

- Point to **[Bellman stage decomposition](https://llorracc.github.io/workspace-course-topics/assignments/bellman-stage-decomposition.html)** (Class 07) and the **Imai–Keane** example notebooks as the immediate next build targets.
- Remind students to keep **LaTeX / REMARK setup** from Class 06 pre-class work moving in parallel where required.
